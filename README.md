## 機能 (features)

実装済み (Implemented)n\
-クラスとしてカードを定義 (Define Card as a class)
-カードのドラッグ (Dragging any card)
-カード同士の衝突判定 (Collision detection between cards)
-終了ボタン (Quit Button)

実装予定 (Planned for implementation)
-各種セル (Various cells)
-複数のカードをまとめてドラッグ (Drag multiple cards at once)
-一度にドラッグできるカードの枚数制限 (Limit on the number of cards that can be dragged at once)

実装悩み中 (Struggling with the implementation)
